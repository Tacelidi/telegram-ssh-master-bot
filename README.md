# Telegram SSH Master Bot

Telegram‑бот для удалённого управления серверами по SSH: запуск команд, перезагрузка, выключение и работа с несколькими серверами из одного чата.

## Возможности

- Подключение к серверам по SSH  
- Хранение списка серверов (БД)
- Выполнение произвольных команд на выбранном сервере
- Перезагрузка и выключение сервера

## Технологии

- aiogram
- SQLite 
- SSH‑подключение 
  
## Структура проекта

  telegram-ssh-master-bot/
  ├── bot.py # Точка входа, инициализация бота и диспетчера
  |   handlers.py # Хэндлеры, логика диалогов и FSM
  |   keyabords.py # Клавиатуры (ReplyKeyboardMarkup)
  |   DB.py # Работа с базой данных (серверы, пользователи и т.п.)
  |   SSH.py # Логика SSH‑подключения и выполнения команд
  └── requirments.txt


## Запуск

1. Клонировать репозиторий
2. Установить зависимости
3. Создать .env и добавить токен бота


## Как это работает

- Пользователь запускает бота (`/start`) и попадает в меню.
- Через кнопки добавляет сервер: имя, адрес, логин, пароль.
- Выбирает сервер и действие:
- отправить команду;
- выключить;
- перезагрузить;
- получить данные.
